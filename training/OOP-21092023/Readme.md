Das ist eine Objektorientierte Programmierung mit PHP. Das Thema: Bank

Die funktionen sind bekanntlich:
  - Girokonto erstellen
  - Sparkonto erstellen
  - Namen vergeben
  - Konto-Nummer vergeben
  - Kontostand abfragen
  - einzahlen
  - auszahlen
  - überweisen

Sonderfunktion:
  Es ist keine funktion "auszahlen" möglich, wenn "kontostand" bei 0 liegt.
